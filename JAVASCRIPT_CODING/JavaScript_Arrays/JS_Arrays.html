JavaScript Arrays

An array is a special variable, which can hold more than one value:

const cars = ["Saab", "Volvo", "BMW"];

Why Use Arrays?

If you have a list of items (a list of flower names, for example), storing the flowers in single variables could look like this:

let flower1 = "Rose";
let flower2 = "Lotus";
let flower3 = "Lilly";
let flower4 = "Sunflower";

However, what if you want to loop through the flowers and find a specific one? And what if you had not 4 flowers, but 400 different flowers?

The solution is an array!

An array can hold many values under a single name, and you can access the values by referring to an index number.
